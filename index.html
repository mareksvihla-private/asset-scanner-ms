How it works

AssetScan uses the Microsoft Graph API to read and write rows in an Excel file stored in your SharePoint Documents library. Staff sign in with their existing school Microsoft 365 account — the same one used for Teams and SharePoint. All asset data is visible to anyone with access to that SharePoint site.

Step 1 — Azure App Registration

This is the one-time technical setup. Ask your school's IT admin if needed — they may be able to do steps 1–5 for you.

1Go to portal.azure.com and sign in with a Microsoft 365 admin account.
2Search for App registrations in the top search bar → click + New registration.
3Name: AssetScan · Supported account types: Accounts in this organisational directory only · Click Register.
4On the Overview page, copy the Application (client) ID — paste it in the Client ID field on the login screen.
5In the left menu click Authentication → + Add a platform → Single-page application.
6Enter your hosted app URL as the Redirect URI, e.g. https://YOURUSERNAME.github.io/asset-scanner. Click Configure.
7Tick both Access tokens and ID tokens under Implicit grant. Click Save.
8In the left menu click API permissions → + Add a permission → Microsoft Graph → Delegated permissions.
9Search for and add: Files.ReadWrite and Sites.ReadWrite.All. Click Add permissions.
10Click Grant admin consent for [school name] → Yes. This allows all staff to use the app without individual permission prompts.

Step 2 — SharePoint

1Go to your school's SharePoint site (e.g. open Teams → your IT channel → Files → open in SharePoint).
2Copy the site URL from the browser address bar — everything up to and including the site name, e.g. https://school.sharepoint.com/sites/IT. Don't include any page paths.
3The app will automatically create AssetRegister.xlsx (or your chosen file name) in the Documents library when you first save an asset.
4The Excel file is visible in Teams under that site's Files tab. Anyone with site access can open, filter, and export the data.

Step 3 — Host the app

1Go to github.com → sign up free → New repository named asset-scanner (Public).
2Upload this HTML file, rename it to index.html, then go to Settings → Pages → Branch: main → Save.
3Your app URL will be: https://YOURUSERNAME.github.io/asset-scanner — share this with staff.
4On iPad/iPhone: open the URL in Safari → Share button → Add to Home Screen — works like a native app.
